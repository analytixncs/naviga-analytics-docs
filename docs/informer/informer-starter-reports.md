## Informer Starter Reports

This page contains a series of ad hoc reports and datasets covering the primary mappings and data.

Each dataset is also prepared for use with AI tools. These datasets include a special Powerscript entry that describes each field and its purpose.

A corresponding JSON download is also available on this page. It combines the dataset field descriptions into a structured format that can be easily ingested by an AI tool.
